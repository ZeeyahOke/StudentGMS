# Sudent Grade Management System

## Overview
This is a console-based application, that allows users to add, search, update, delete and analyze students and their grades.

## Requirements
To run this application, you need:
- .NET SDK installed (version 6.0 or higher)

You can check if .NET is installed by running the command `dotnet --version` in your terminal.

## How to Run
- Step 1: Clone this repo by using `git clone https://github.com/ZeeyahOke/StudentGMS`
- Step 2: Navigate into the repo by using the command `cd StudentGMS`
- Step 3: In your terminal, run the command `dotnet build` to build the project
- Step 4: Run the application using the command `dotnet run`
